Static image assets for the Delivery Pods site, served directly by Vercel from this repo.

A file placed here at images/NAME is served at https://wrg-delivery-pods.vercel.app/images/NAME
Reference it in the HTML with a root-relative path: /images/NAME

To add an image: upload it into this images/ folder (or attach it in chat and it gets committed here), then reference it as /images/FILENAME in the page(s).
